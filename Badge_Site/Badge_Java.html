ランキングの決め方について

GithubCI/CD

  1. Github actions

    対象のサイトが正しく動いたかどうかを表している

  2. Github releases

    最終リリース日を記入している。

  3. Github LICENSE

    このリポジトリが使用している主なライセンスをBadgeにしている。

CI/CD

  1. circleci.com

    このサイトはcircleCIという企業向けのサポートとサービスを提供しているCI/CDである。主にLinux,macOS,Android、Windowsで利用できる。

  2. ci.appveyor.com

    このサイトはAppVeyorというCI/CDである。主にWindows環境に特化しており、主にWindows、最近のアップデートでLinux、macOSのビルドができるサイトとなっている。

テストカバレッジ

  1. coveralls.io

    Coverallsは、ソフトウェアのテストカバレッジを追跡・可視化するサービスであり、CIツールと統合し、コードの変更ごとにカバレッジ情報を自動更新できる。

  2. codecov.io

    Codecovは、テストカバレッジの計測結果を可視化したり、ソフトウェアのバグなどを検出することができる。ソフトウェアの品質向上に役立つ。

  3. www.codacy.com

    Codacyは、コード・レビューを自動化し、各コミットおよびプル・リクエストのコード品質を監視するサイトである。

コード品質

  1. www.javadoc.io

    Java言語によって記述されたクラスライブラリやアプリケーションに関するパッケージやクラスやメソッドに関するHTML形式のAPI仕様書のサイトである。

  2. lgtm.com

    これはLGTMというコードを分析するためのツールで、CodeQLを使用しソースコードに対してクエリを投げることで脆弱性、不具合のあるコードを検査することができる。

  3. isitmaintained.com

    isitmaintainedはオープンソフトウェアのopenissueに基づいてプロジェクトがメンテナンスされているかを確認するサイトです。

ライセンス

  1. www.apache.org

    ApacheとはオープンソースのWebサーバーソフトである。様々なOSで動作でき、最も人気のあるウェブサーバーソフトウェアである。

  2. opensource.org

    Open Source Initiativeの公式ウェブサイトであり、主にオープンソースライセンスの承認と教育活動を行っている。

  3. www.gnu.org

    フリーソフトウェアのみでUNIX風のOS、ソフトウェアを開発、公開するプロジェクトである。

外部サイト

  1. gitter.im
  2. X

リリースバージョン

  1. maven-badges.herokuapp.com

  2. badge.fury.io

    3. badge.furyとはRubyGems, PyPI, npmなどのバージョンバッジを生成することができる。 また新しいバージョンにも対応し、自動的に更新される。

  3. zenodo.org

    Zenodoとは研究データリポジトリである。様々な分野の研究者でも50GBまでファイルをアップロードすることができる。

セキュリティー

  1. Sonarcloud.io

    SonarCloudとはCI/CDワークフローで動く静的コード解析ツールである。Github上でのPull Request時やPush時にソースコードを解析することができる。

  2. Api.Securityscorecards.dev

プラットフォーム

  1. Android-Arsenal.com

  2. Play.Google.com

    Google Play Storeのウェブ版のサイトである。Googleによって運営されているAndroidデバイス向けの様々な娯楽サービスのサイトである。

  3. Developer.Android.com

開発ツール

  1. search.maven.org
  2. travis-ci.org

    Travis-ciとは分散型の継続的インテグレーション サービスであり、設定ファイルひとつでテストを自動化できる。

  3. jitpack.io

デプロイ環境

  1. gitpod.io

    gitpodはGithubなどのリポジトリに連動し、プルリクエストから即座に完全な開発環境を起動することができるサイトである。

  2. mybinder.org

    mybinderとはGitHub 上のJupyter Notebooks をクラウドで実行することができるサイトである。

  3. dev.azure.com

    dev.azureとはMicrosoft社が提供しているサーバーやネットワークなどのITインフラをクラウド上で提供するサービスです。
