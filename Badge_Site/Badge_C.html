人気BadgeC

ランキングの決め方について

GithubCI/CD

  1. Github actions

    対象のサイトが正しく動いたかどうかを表している

  2. Github releases

    最終リリース日を記入している。

  3. Github LICENSE

    このリポジトリが使用している主なライセンスをBadgeにしている。

CI/CD

  1. ci.appveyor.com

    このサイトはAppVeyorというCI/CDである。主にWindows環境に特化しており、主にWindows、最近のアップデートでLinux、macOSのビルドができるサイトとなっている。

  2. circleci.com

    このサイトはcircleCIという企業向けのサポートとサービスを提供しているCI/CDである。主にLinux,macOS,Android、Windowsで利用できる。

  3. oss-fuzz-build-logs.storage.googleapis.com

    このサイトはOSS-Fuzzといい、Google製のCI/CDである。オープンソフトウェアに対する脆弱性のテストを行える。

テストカバレッジ

  1. codecov.io

    Codecovは、テストカバレッジの計測結果を可視化したり、ソフトウェアのバグなどを検出することができる。ソフトウェアの品質向上に役立つ。

  2. coveralls.io

    Coverallsは、ソフトウェアのテストカバレッジを追跡・可視化するサービスであり、CIツールと統合し、コードの変更ごとにカバレッジ情報を自動更新できる。

コード品質

  1. scan.coverity.com

    これはCoveityといい、ソースコードの不具合や脆弱性を検出する静的解析ツールである。

  2. bestpractices.coreinfrastructure.org

    これはBestPracticesといいプロジェクトがBadgeの合格基準を満たしていると、Badgeをつけることができる。

  3. lgtm.com

    これはLGTMというコードを分析するためのツールで、CodeQLを使用しソースコードに対してクエリを投げることで脆弱性、不具合のあるコードを検査することができる。

ライセンス

  1. app.fossa.io

    これはFossaというセキュリティの自動監視・管理ツールであり、プロジェクトの依存関係を分析し、ライセンス違反や、セキュリティリスクを識別する。

  2. www.gnu.org

    フリーソフトウェアのみでUNIX風のOS、ソフトウェアを開発、公開するプロジェクトである。

外部サイト

  1. gitter.im
  2. X
  3. Youtube

リリースバージョン

  1. repology.org

    Repologyは多様なパッケージリポジトリのソフトウェアバージョン情報を一元表示するサービスでユーザーが特定のソフトウェアのバージョン状況を各ディストリビューションで確認できる。

セキュリティー

  1. sonarcloud.io

    SonarCloudとはCI/CDワークフローで動く静的コード解析ツールである。Github上でのPull Request時やPush時にソースコードを解析することができる。

プラットフォーム

  1. snapcraft.io

    Snapcraftとは開発者によって作られたプログラムをSnappy用にSnapフォーマットでパッケージングするツールである。

開発ツール

  1. travis-ci.org

    Travis-ciとは分散型の継続的インテグレーション サービスであり、設定ファイルひとつでテストを自動化できる。

  2. travis-ci.com
  3. colab.research.google.com

    colabとは誰でもブラウザ上で Python を記述、実行できるため、機械学習、データ分析、教育に特に適している。

デプロイ環境

  1. dev.azure.com

    dev.azureとはMicrosoft社が提供しているサーバーやネットワークなどのITインフラをクラウド上で提供するサービスです。

  2. aws.amazon.com　

    Amazon社が提供しているクラウドコンピューテングのサービスである。ストレージやOSなどをネットワーク上で利用することができます。
